1. **p2p**
   - เครื่องมือแปลงโค้ดระหว่างไพทอนและปิยะทอน
   - ติดตั้งพร้อมกับปิยะทอน
   - [คำอธิบายการใช้งาน](../tutorial/migration.md)

2. **IDE Plugins**
   - Visual Studio Code
   - PyCharm
   - [รายละเอียดการติดตั้ง](../getting_started/configuration.md)
