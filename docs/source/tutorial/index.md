# บทเรียนปิยะธอน

ยินดีต้อนรับสู่บทเรียนปิยะธอน ส่วนนี้จะแนะนำวิธีการใช้งานภาษาปิยะธอนตั้งแต่ระดับพื้นฐานไปจนถึงขั้นสูง

## วิธีการเรียนรู้

บทเรียนแต่ละบทจะประกอบด้วย

- ทฤษฎีและคำอธิบาย
- ตัวอย่างโค้ด
- แบบฝึกหัด
- คำถามท้ายบท
