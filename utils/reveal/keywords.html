Control Flow - Conditionals

Python

    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")

Piyathon

    พิมพ์("บวก")
อื่นถ้า x < 0:
    พิมพ์("ลบ")
อื่น:
    พิมพ์("ศูนย์")

Control Flow - Exception Handling

Python

try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"Error: {e}")
finally:
    print("Done")

def check_value(x):
    if x < 0:
        raise ValueError("Negative")

Piyathon

ลอง:
    ผลลัพธ์ = 10 / 0
ยกเว้น ZeroDivisionError เป็น e:
    พิมพ์(f"ข้อผิดพลาด: {e}")
สุดท้าย:
    พิมพ์("เสร็จสิ้น")

นิยาม ตรวจสอบค่า(x):
    ถ้า x < 0:
        ยก ValueError("ติดลบ")

Logical Operators

Python

if x > 0 and y > 0:
    print("Both positive")

if x is None or y is None:
    print("One is None")

if not ready:
    print("Not ready")

if x in [1, 2, 3]:
    print("Found")

Piyathon

ถ้า x > 0 และ y > 0:
    พิมพ์("ทั้งคู่เป็นบวก")

ถ้า x คือ ไม่มีค่า หรือ y คือ ไม่มีค่า:
    พิมพ์("หนึ่งในนั้นเป็นไม่มีค่า")

ถ้า ไม่ พร้อม:
    พิมพ์("ยังไม่พร้อม")

ถ้า x ใน [1, 2, 3]:
    พิมพ์("พบแล้ว")
